# Наш DevOps проект

## Описание проекта
Проект направлен на демонстрацию базовых практик DevOps: контейнеризации приложения, настройки окружения и автоматизации процесса запуска. В рамках проекта используется Linux-среда и Docker для развертывания и тестирования приложения.

Основная цель — упростить процесс разработки, сборки и запуска приложения, а также обеспечить стабильную работу в разных окружениях.
